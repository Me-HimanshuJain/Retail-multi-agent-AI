# Code of Conduct

We expect all contributors to behave professionally and respectfully. Harassment, discrimination, abuse, or exclusionary behavior will not be tolerated.

## Standards

- Be respectful and constructive.
- Focus on the technical problem.
- Give and receive feedback in good faith.
- Protect privacy and security.

## Enforcement

Project maintainers may remove, edit, or reject content that violates this code of conduct.
